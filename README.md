## About

Portfolio website for Zerotouch — an agency that builds automation systems, operational dashboards, and websites for businesses that want to run without constant human intervention.

## Stack

- Pure HTML + CSS + Vanilla JS
- GSAP (ScrollTrigger animations)
- Google Fonts (Instrument Serif + DM Sans)
- Zero frameworks. Zero dependencies to install.

## Local Development

Just open `index.html` in your browser. No build step needed.

```bash
# Or use a local server
npx serve .
```

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/zerotouch`

## Custom Domain (zerotouch.co)

1. Add a `CNAME` file with your domain: `zerotouch.co`
2. In your domain registrar (Hostinger), point DNS:
   - A record → `185.199.108.153`
   - A record → `185.199.109.153`
   - A record → `185.199.110.153`
   - A record → `185.199.111.153`
3. Enable **Enforce HTTPS** in GitHub Pages settings
